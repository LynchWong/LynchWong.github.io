变量声明

Go语言使用关键字var来声明变量，变量类型放在变量名之后，如下所示：

var v1 int
var v2 string 
var v3 [10]int          // 数组
var v4 []int            // 数组切片
var v5 struct {
    f int 
}
var v6 *int             // 指针
var v7 map[string]int   // map,key为string类型,value为int类型
var v8 func(a int) int
