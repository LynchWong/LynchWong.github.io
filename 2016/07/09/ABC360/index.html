ABC360[手动置顶]

Posted on 2016-07-09   |     |  

说明：这里是iOS版本的下载安装。

安装前提

  • 连接公司内网，外网无法下载成功
  • 使用手机 Safari 浏览器打开本网址进行下载

版本选择

运行环境 版本 点击下载
开发环境 v3.1.0 下载
内测环境 v3.1.0 下载
线上环境 v3.1.0 下载
Shadow环境 v3.1.0 下载

环境说明

  1. 开发环境：开发人员使用的环境。
  2. 内测环境：跟线上环境是一样的，是线上环境的一个镜像。
  3. 线上环境：就是正式环境，跟在App Store下载的版本没有区别，下载后请慎重操作。
  4. Shadow环境：模拟线上环境。

版本说明

  • v3.1.0 当前开发的最新版本，领先正式版本。

PS：如果你不明白环境和版本，不知道该下哪一个，请咨询测试大神：Ices。

二哥专用版

你不是二哥就不要下载这个版本了，请自动对号入座。

  • 二哥专用VIP定制版

安装步骤

根据上面的列表选择适合自己的环境和版本，然后点击下载，点击后的效果如下图所示：

点击安装就可以了。

点击安装后返回桌面，界面大概如下所示：

这两种情况都是正常的，情况一能明显看到App在安装。情况二应该是网络导致的问题，但是导航栏能看到网络请求，安装好后就会出现App的Logo。

信任开发者

安装完成后点击打开App会提示你信任开发者，这是最后一步。会出现如下界面：

提示的是不受信任的企业开发者，这时候就要去设置里面进行修改一下，以iOS 9为例。

首先进入到设置->通用里面，然后找到设备管理：

进入到设备管理，然后点击我们的企业证书：

然后点击信任我们的企业证书：

然后选择信任：

最后打开App，如下图所示：

结语

至此就安装完成了，如果安装有什么问题请仔细阅读文档，文档已经描述的很清楚了。记住安装的前提是要链接公司的内网。

另外安装的 IPA 包是存储在公司内网的，所以一定要链接公司内网才能安装成功。即使外网用户看到这篇文章想要安装也是无法成功的，这样避免了公司外网用户直接获取到 IPA 包。
